More information

My projects are made mainly in the C-family, sometimes Java or Lua, sometimes just Minecraft plugins. I prioritize projects that are useful to me and others.


Source code can be found on my GitHub

Support?

Software support may come, but it is not guaranteed. If needed, refer to the README.md files on the GitHub repository for help. These projects are simple to use, just download it and run it. If it's a Minecraft plugin, place it in your plugins folder and restart your server.

VMU

VMU (VeinMinerUltimate) is an advanced Minecraft server plugin, intended to allow players to mine veins of blocks more efficiently, along with enchantment support, including 'Silk Touch' & 'Fortune.' VMU is toggleable through in game commands, the host command is '/vmu.'

CLU

CLU (ChunkLoadersUltimate) is an advanced Minecraft server plugin, intended to allow players to keep chunks loaded while not nearby. It has a unique chat purchasing system, which has a customizable price via the 'config.yml' file. Default chunk loader block price is 3 diamonds.

OPS

OPS (OnePlayerSleep) is a Minecraft plugin intended to simply let players sleep when only 1 person is sleeping to skip the night.


This plugin will recieve future updates to enhance it's functionality.

Minecraft Plugin Information

My Minecraft plugins are made using the Bukkit API for 1.16.5, meaning they are compatible with Bukkit, Spigot, Paper, and Purpur servers running 1.16 and up. It is not guaranteed the plugins work on older versions than 1.16.

Program Information

For clarification, my programs only work on Windows. I've only been able to test Windows 11, not older versions. Functionality for older versions of Windows is not guaranteed, and may result in unexpected behavior or errors.
